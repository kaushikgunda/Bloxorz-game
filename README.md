# ASSIGNMENT 2 - COMPUTER GRAPHICS

## Building

1. Run `make` to compile
2. Run `sample2D`

## Controls

- **UP**, **RIGHT**, **LEFT**, **RIGHT**
- **C** to toggle camera
- **MOUSE SCROLL** to zoom in and out in helicopter mode
- **MOUSE RIGHT** to click and drag camera in helicopter mode

## Rules

- Complete levels in least number of moves
